## Research Data App (Flutter Frontend)

This repository contains the Flutter mobile frontend for the Research Data App.

### Prerequisites
- Flutter SDK 3.x
- Dart SDK 2.19.x
- A running FastAPI backend at http://localhost:8000/api

### Setup
```bash
cd /home/yutw/project/md_templates/results/research_data_app_flutter/frontend
flutter pub get
```

### Run
```bash
flutter run
```

The app will display a list of research data fetched from the backend.
